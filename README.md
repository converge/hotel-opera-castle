### Hotel Management

This is a fictitious project, to try new things. Feel free to join, the purpose of this projects is to have fun.

---

#### Concept:

It's a Hotel Management System, the version 1 should contain these functionalities:

- Clients Management
- Rooms Management
- Check In
- Check Out

#### Stack

- NodeJS
- PostgreSQL
- libs: jest, db-migrate

#### Database ERM

clients:  
.id  
.first name  
.last name  
.created  
.modified  

client_room:  
.id  
.client_id  
.room_id  
.check_in  
.check_out  
.created  
.modified  

rooms:  
.id  
.price  
.description  
.client_id  
.created  
.modified
